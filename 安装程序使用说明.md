# 视频字幕去除器 - Windows 安装程序

## 📦 安装程序信息

**文件名**: `VideoSubtitleRemover-Setup-v1.4.0.exe`
**大小**: 732 MB
**版本**: v1.4.0
**发布日期**: 2026-04-10

## 🚀 安装步骤

### 1. 运行安装程序

双击 `VideoSubtitleRemover-Setup-v1.4.0.exe`

### 2. 接受 UAC 提示

Windows 会弹出用户账户控制（UAC）提示：
```
您要允许此应用对您的设备进行更改吗？
程序名称: 视频字幕去除器
发布者: YaoFANGUK
```
点击 **"是"** 继续

### 3. 按照安装向导完成安装

- **选择安装位置**: 默认为 `C:\Program Files\视频字幕去除器\`
- **选择开始菜单文件夹**: 默认为 "视频字幕去除器"
- **选择附加任务**:
  - ✓ 创建桌面快捷方式
  - ✓ 添加到开始菜单
  - ✓ 添加到启动文件夹

### 4. 完成安装

- 安装完成后可以选择立即启动程序
- 程序会自动请求管理员权限（UAC 提示）
- 首次运行可能需要 1-2 分钟初始化 AI 模型

## ✨ 安装程序功能

### ✅ 已包含的功能

## 🎯 使用方式

### 方式一：桌面快捷方式（推荐）

双击桌面上的 **"视频字幕去除器"** 快捷方式

### 方式二：开始菜单

1. 点击 Windows 开始按钮
2. 在程序列表中找到 **"视频字幕去除器"**
3. 点击启动

### 方式三：直接运行

1. 打开安装目录：`C:\Program Files\视频字幕去除器\`
2. 双击 `VideoSubtitleRemover.exe`

## 🔧 卸载程序

### 方法一：通过 Windows 设置

1. 打开 **设置** → **应用** → **应用和功能**
2. 搜索 **"视频字幕去除器"**
3. 点击 **卸载** 按钮
4. 按照卸载向导完成卸载

### 方法二：通过开始菜单

1. 在开始菜单中找到 **"视频字幕去除器"**
2. 右键点击 → **卸载**
3. 按照卸载向导完成卸载

### 方法三：直接运行卸载程序

1. 打开安装目录：`C:\Program Files\视频字幕去除器\`
2. 找到 `unins000.exe`（卸载程序）
3. 双击运行，按照向导完成卸载

## 💡 系统要求

### 最低要求

- **操作系统**: Windows 10/11 (64位)
- **内存**: 8 GB RAM（推荐 16 GB）
- **磁盘空间**: 2 GB 可用空间
- **显卡**: 支持 CUDA 11.8/12.6 的 NVIDIA GPU（可选）

### 推荐配置

- **操作系统**: Windows 11 (64位)
- **内存**: 16 GB RAM 或更多
- **磁盘空间**: 4 GB 可用空间
- **显卡**: NVIDIA GPU with 4GB+ VRAM
- **CPU**: Intel i5 或 AMD Ryzen 5 或更高

## ⚠️ 注意事项

### 首次运行

- **初始化时间**: 首次运行可能需要 1-2 分钟初始化 AI 模型
- **UAC 提示**: 每次启动都会弹出 UAC 提示，需要管理员权限
- **网络连接**: 程序不需要网络连接，但更新功能需要网络

### 权限要求

程序需要管理员权限的原因：
- 访问受保护的文件系统区域
- 系统级操作和进程管理
- FFmpeg 等工具的某些功能
- 更好的兼容性

### 存储空间

- **安装程序大小**: 732 MB
- **安装后大小**: 约 1.8 GB
- **运行时内存**: 约 2-4 GB
